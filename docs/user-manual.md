This guide explains how to use the Online Bookstore System for both customers and administrators.

---

## 1. Customer Guide

### 1.1 Registration and Login
- Open the Register page
- Enter email and password
- Login using registered credentials

---

### 1.2 Browsing and Searching Books
- Browse books by category
- Use search bar to find books by title, author, or ISBN
- Click a book to view details

---

### 1.3 Shopping Cart
- Click "Add to Cart" on a book
- View cart and update quantities
- Remove items if needed

---

### 1.4 Checkout and Orders
- Proceed to checkout
- Enter shipping details
- Complete payment
- View order confirmation
- Track orders in "My Orders"

---

## 2. Admin Guide

### 2.1 Login
- Login using admin credentials

---

### 2.2 Catalog Management
- Add new books
- Edit existing books
- Remove books from catalog

---

### 2.3 Order Management
- View all orders
- Update order status (Processing, Shipped, Delivered)
- Review order history
